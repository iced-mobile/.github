## Iced Mobile

This organization is dedicated to bringing first-class mobile support to Iced. Currently, only Android support is being developed.

### Where to start

We have a [book](https://github.com/iced-mobile/book) that cover all you need to know about developing an Iced app on Android.

We only have an example app [here](https://github.com/iced-mobile/todolist) which provide a good starting point.

### Note 

This project is not affiliated with the [Iced](https://github.com/iced-rs/iced) project. The maintainer of Iced has clearly stated that he is not interested in supporting mobile platforms (see https://github.com/iced-rs/iced/issues/302).
